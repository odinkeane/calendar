# Календарь для отслеживания своих активностей

## Установка

```bash
npm install
cd frontend && npm install
```

## Запуск 
```bash
npm run dev
```

## Подтягивание файлов

В данной версии подтягивание файлов формата `json` отсутствует. Можно их ложить в папку jsons. Пример хранения файлов с теми ключами, которые используются в календаре:
```json
[
    {
        "date": "YYYY-MM-DD",
        "time": [
            "HH:MM-HH:MM"
        ],
        "theme": "Название темы",
        "title": "Название курса"
    },
]
```
